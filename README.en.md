### Run the project

Windows: PowerShell
```
In the File explorer, right click on `Windows.ps1`, and left click on `Running in PowerShell`.
```

Linux: Bash or Mac: Terminal
```
Open the bash or termianl, then input `bash ./Linux&Mac.sh` or just `./Linux&Mac.sh`, and Enter it.
```
